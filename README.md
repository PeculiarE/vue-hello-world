## Project setup
```
run cmd as admin
npm install vue (this is to install vue globally on your system)
npm install -g @vue/cli (this is to install cli globally on your system)
vue --version (this is to check the vue version installed)
create a folder for the project and change the directory of the cmd to the created folder:
from cmd admin (C:\WINDOWS\system32>) to (c:\Users>) using (C:\WINDOWS\system32>cmd /k cd c:\Users)
then use (c:\Users> cd Peculiar/Documents/Enyata Academy/Vue) to move to the desired directory(c:\Users\Peculiar\Documents\Enyata Academy\Vue>)

continuing with cmd or porting to git bash:
vue create hello-world (this is to create a new vue project titled hello-world)
? Please pick a preset: Manually select features
? Check the features needed for your project: Choose Vue version, Babel, Router, Linter
? Choose a version of Vue.js that you want to start the project with 2.x
? Use history mode for router? (Requires proper server setup for index fallback in production) No
? Pick a linter / formatter config: Airbnb
? Pick additional lint features: Lint on save
? Where do you prefer placing config for Babel, ESLint, etc.? In dedicated config files
? Save this as a preset for future projects? Yes
? Save preset as: VueTemplate

cd hello-world (this is after the project has been created successfully and changes the directory to the created project)
```
